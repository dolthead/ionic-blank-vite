# Vite with the Ionic blank starter

Fork and clone to dive in to your project.

## Start it up

> `npm run serve`

## Run unit and e2e tests

> `npm run test`
>
> `npm run coverage`
> 
> `npm run e2e`

## Prod build

> `npm run build`

## Run as a PWA

> `npm run serve:pwa`
